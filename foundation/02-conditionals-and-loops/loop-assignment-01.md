# Looping Statements
1. Test if something will trigger loop
2. Perform action
3. Update step, repeat #1
## For
```py
for i in range(10): # Test condition
    # do something 10 times
```
## While
```py
i = 0
while(i < 10): # Test condition
    # do something 10 times
    i+=1
```

## Do-While
```java
// Not supported by Python
// In Java
do{
    // do something
    i++;
}while(i<10); // Test condition
```
